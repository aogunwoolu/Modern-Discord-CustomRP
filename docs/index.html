What it does

Everything you need to control your Discord status

One app to manage all your Discord Rich Presence activities — no coding, no config files, no extensions.

Preset Library
Hundreds of pre-configured known apps — games, browsers, music players, IDEs and more. One click to load and connect.
Auto-Detect
Scans all running processes and matches them against the known-app library. Hit Use and the editor is pre-filled instantly.
Live Auto-Update
Polls a process at a configurable interval, reads the window title or browser URL, and pushes it to Discord in real time.
Favicon as Small Image
When browsing, the current site's icon is automatically fetched and displayed as the small image badge on your presence.
Multiple Presences
Each preset can use its own Discord Application ID so they stack in Discord's Activity Shelf without replacing each other.
Template Variables
Use {title}, {url}, {host}, {path} and more in any text field to inject live data from the tracked process.
Activity Shelf Timestamps
Show elapsed time, remaining time, or a custom start/end timestamp. Discord renders the timer automatically.
Startup Restoration
Active presences are saved to disk and reconnected automatically the next time you open the app — no manual reconnecting needed.

Get Started

Up and running in under a minute

1
Download & Extract
Grab the latest .zip from the Releases page. Extract CustomRP.Modern.exe anywhere. Install the .NET 10 Desktop Runtime if prompted.
2
Pick an App or Start Fresh
The Preset Library opens by default. Search for any app and hit Load, or open the Editor tab to configure something custom.
3
Click Connect
A Discord Application ID is pre-filled as the default. Click Connect — your presence goes live immediately in Discord.
4
(Optional) Enable Auto-Update
Turn on Auto-update in the editor. Select Browser URL strategy and enable Use website favicon as small image to get a live favicon alongside the page title.
